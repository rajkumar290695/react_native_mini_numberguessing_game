🎮 Guess My Number

A fun and interactive mobile game built with React Native and Expo where the phone tries to guess the number chosen by the user using higher/lower hints.

📱 Screens Included
Start Game Screen
Opponent Guess Screen
Guess History List
Game Over Screen
Validation Alert Popup



✨ Features
🎯 User selects a number between 1–99
🤖 Phone intelligently guesses the number
⬆️⬇️ User gives Higher or Lower hints
📜 Displays guess history with round count
🚫 Detects invalid hints and shows warning alert
🔄 Restart and play unlimited times
🎨 Clean modern UI with custom styling


🛠️ Tech Stack
React Native
Expo
JavaScript
React Hooks (useState, useEffect)
Custom Components
Responsive Mobile Design

🚀 Installation
git clone <your-repo-link>

cd guess-my-number

npm install

npx expo start
